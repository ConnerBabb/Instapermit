# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do NOT open a public GitHub issue.**
2. Use GitHub's [private vulnerability reporting](https://github.com/ConnerBabb/Instapermit/security/advisories/new).

I will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Security Measures

- Dependabot monitors dependencies for known vulnerabilities
- Secret scanning with push protection is enabled
- CI runs linting and tests on every push and pull request
